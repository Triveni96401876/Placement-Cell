# Placement Cell Management System

## Project Overview
A web-based application for managing campus placements, connecting students with administrators and companies.

## Tech Stack
*   **Frontend**: HTML, CSS (Vanilla/Responsive).
*   **Backend**: Java (Servlets/JSP), JDBC.
*   **Database**: MySQL.

## Documentation
*   [Flowchart & Working Process](./FLOWCHART.md): Detailed explanation of the system diagrams and user flows.

## Directory Structure
*   `backend/`: Java source code (to be refactored for JDBC/Servlet focus).
*   `frontend/`: (Planned) To contain the HTML/CSS pages.
*   `database/`: (Planned) SQL scripts.

## Key Features
1.  **Student Module**: Registration, Profile, Resume Upload, Job Application.
2.  **Admin Module**: Student Management, Job Postings (Circulars), Hiring Status.
